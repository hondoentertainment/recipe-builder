Connect Google Photos

Load your real photo library locally, or upload recipe photos directly in the picker. Extracted recipes are saved in this browser and appear in the recipe library.

Option A — Upload photos (works now)

  1. Open the photo picker
  2. Click Upload photos and choose recipe images
  3. Select photos and click Extract recipes
  4. Review, favorite, edit, print, or share in the recipe library
Open picker Browse recipes

Option B — Google Photos API (local setup)

  • 1. Create a Google Cloud project
  • 2. Enable Photos Library API
  • 3. Create OAuth credentials (Desktop app)
  • 4. Download JSON → save as credentials/client_secret.json
  • 5. Run python connect_google_photos.py and sign in
  • 6. Run python run_all.py to select photos and build recipes

After OpenAI quota is available
