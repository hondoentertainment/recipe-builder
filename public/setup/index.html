Connect Google Photos

Load your real photo library locally, or upload recipe photos directly in the picker.

Option A — Upload photos (works now)

  1. Open the photo picker
  2. Click Upload photos and choose recipe images from your computer
  3. Select photos and click Extract recipes
Open picker

Option B — Google Photos API (local setup)

  1. Create a Google Cloud project and enable the Photos Library API
  2. Create OAuth credentials (Desktop app) and download client_secret.json
  3. Save it to credentials/client_secret.json in this project
  4. Run python connect_google_photos.py and complete sign-in
  5. Run python run_all.py to select photos and build recipes
Tip: Your IMG photos are already in this project folder. Run python generate_from_img.py --skip-convert locally to re-process them, then python export_web_recipes.py && npm run build to refresh the recipe library.
